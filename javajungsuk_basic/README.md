#### 03. 변수의 선언과 저장 `2-4`, `2-5`
- 변수란? 하나의 값을 저장할 수 있는 저장공간
- 변수 선언 방법: `변수타입 변수이름`
  ```java
  int x;  // 정수(integer)를 저장하기 위한 변수 x 선언 = x라는 저장공간(변수) 생성
  x = 5;  // 변수 x에 5 저장
  int x = 5;  // 한번에 선언 및 저장 가능
  ```
- 변수 장점
  - 아래와 같이 값을 변경해줘야 할 때, 일일히 바꿔주는 대신 저장하는 값(대입값)만 바꿔주면 된다.
  ```java
  // 변수 사용 x
  System.out.println(5+3);
  System.out.println(5-3);
  System.out.println(5*3);
  System.out.println(5/3);
  
  // 변수 사용 o
  int x = 5;  // 변수에 다른 값을 저장하기만 하면 된다.
  int y = 3;  // 변수에 다른 값을 저장하기만 하면 된다.
  System.out.println(x+y);
  System.out.println(x-y);
  System.out.println(x*y);
  System.out.println(x/y);
  ```

#### 04. 변수의 타입 `2-6`
- 변수 선언 시, 변수에 저장할 값의 종류에 맞는 변수 타입 선택
- 변수의 타입:
- ![image](https://user-images.githubusercontent.com/90819869/166177100-dbc4135e-f990-4c2c-9a0f-006ea93ff5f9.png)
  - **int**, long: 정수(integer)를 저장하기 위한 타입(20억이 넘을 땐 long)
  - float, **double**: 실수(floating-point number)를 저장하기 위한 타입 (float는 오차없이 7차리, double은 15자리)
  - char: 문자(character)를 저장하기 위한 타입
  - String: 여러 문자(문자열, string)를 저장하기 위한 타입
  ```java
  // 대표 4가지
  
  int x = 100;        // 정수(integer)를 저장할 변수의 타입은 int로 한다.
  double pi = 3.14;   // 실수를 저장할 변수의 타입은 double로 한다.
  char ch = 'a';      // 문자(1개)를 저장할 변수의 타입은 char로 한다.
  String str = "abc"; // 여러 문자(0~n개)를 저장할 변수의 타입은 String으로 한다.
  ```

#### 05. 상수와 리터럴
- 상수
  - 변수처럼 '값을 저장할 수 있는 공간'이지만, 변수와 달리 한 번 값을 저장하면 변경 x
  - 변수의 타입 앞에 `final`을 붙여준다.
  - 이름은 모두 대문자로 하며, 여러 단어로 이루어져 있는 경우 '\_'로 구분한다.
  ```java
  // 예시
  
  final int MAX_VALUE;  // 정수형 상수 MAX_VALUE 선언
  MAX_VALUE = 100;      // OK. 상수에 처음으로 값 저장
  MAX_VALUE = 200;      // ERROR. 상수에 저장된 값 변경 불가
  ```
- 리터럴
  - 기존의 '상수'인 값들 (예: 12, 123, 3.14, 'A' 등)
- 구분
  ```java
  int year = 2014;  // year = 변수, 2014 = 리터럴
  final int MAX_VALUE = 100;  // MAX_VALUE = 상수, 100 = 리터럴
  ```
- |분류|설명|
  |:--:|:--:|
  |변수(variable)|하나의 값을 저장하기 위한 공간|
  |상수constant)|값을 한 번만 저장할 수 있는 공간|
  |리터럴(literal|그 자체로 값을 의미하는 것|

#### 06. 리터럴의 타입과 접미사
- |종류|리터럴|접미사|
  |:--:|:--:|:--:|
  |논리형|false, true|없음|
  |정수형|123, 0b0101, 077m 0xFF, 100L|L|
  |실수형|3.14, 3.0e8, 1.4f, 0x1.0p-1|f, d|
  |문자형|'A', '1', '\n'|없음|
  |문자형|"ABC", "123", "A", "true"|없음|
- 정수형: 접미사(l 또는 L) 존재 시 Long 타입, 없을 시 int 타입의 리터럴 (기본 리터럴 `int`)
- 실수형: 접미사 'f'는 float 타입, 'd'는 double 타입 (기본 리터럴 `double`이므로 'd'는 생략 가능)
- 그 외
  ```java
  // 10진수 외(2, 8, 16진수) 표현법
  int octNum = 010;   // 8진수 10, 10진수로 8    => 접두사 '0' 用
  int hexNum = 0x10;  // 16진수 10, 10진수로 16  => 접두사 '0x' 또는 '0X' 用
  
  // 구분자(_)로 큰 숫자 표기 (JDK1.7 이상)
  long big = 100_000_000_000L;      // long big = 100000000000L;
  long hex = 0xFFFF_FFFF_FFFF_FFFFL // long hex = 0xFFFFFFFFFFFFFFFFL;
  
  // 접미사 'f'와 'd' 차이
  float pi = 3.14f;     // 접미사 f 대신 F 사용 가능, 생략 불가
  double rate = 1.618d; // 접미사 d 대신 D 사용 가능, 생략 가능
  ```

#### 07. 문자 리터럴과 문자열 리터럴
- 문자 리터럴 (`char타입`): 작은따옴표(' ')로 문자 하나를 감싼 것. 빈 문자 허용 x
- 문자열 리터럴 (`String타입`): 큰따옴표(" ")로 두 문자 이상을 감싼 것. 빈 문자열 허용 o
- 예시
  ```java
  // 입력 문자 개수 차이
  char ch = 'J';        // char ch = 'Java'; 불가
  String name = "Java"; // 변수 name에 문자열 리터럴 "Java"를 저장
  
  // 빈 공간 입력 차이
  String str = "";  // OK. 내용이 없는 빈 문자열
  char ch = '';     // ERROR. '' 안에 반드시 하나의 문자 필요
  char ch = ' ';    // OK. 공백 문자(blank)로 변수 ch 초기화
  
  // String 표현
  String name = new String("Java"); // String 객체 생성
  String name = "Java";             // 간단화 한 것
  ```

#### 08. 문자열 결합 `2-7`
- 숫자 외에도, 문자열을 합칠 때 덧셈(+) 사용 가능
- 숫자를 문자열로 변환할 때는, 아무런 내용도 없는 빈 문자열("")을 더해주면 된다.


#### 09. 두 변수의 값 바꾸기 `2-8`
- 두 변수에 담긴 값을 바꾸려면, 값을 임시로 저장할 변수가 하나 더 필요하다.
  ```java
  int x = 10;
  int y = 20;
  
  int tmp;  // 임시로 값을 저장하기 위한 변수(빈 컵 역할)
  
  tmp = x;  // ① x의 값을 tmp에 저장
  x = y;    // ② y의 값을 x에 저장
  y = tmp;  // ③ tmp에 저장된 값을 y에 저장  
  ```

      - 연산에 사용되는 것은 모두 기본형 변수
  2. 참조형(
      - 값이 저장되어 있는 주소(memory address)를 저장

#### 12. printf를 이용한 출력
- `printf()`
  - 같은 값이라도 다른 형식으로 출력하고 싶을 때 用
  - '지시자(sprcifier)'를 통해 변수의 값을 여러 형식으로 변환하여 출력
  - 출력하려는 값의 개수, 순서와 지시자의 개수, 순서는 일치해야 한다. (개수 제한 x)
  - `println()`과 달리, 출력 후 줄바꿈을 하지 않으므로 별도의 지시자(`%n`)를 넣어줘야 한다.
- 지시자
  - |지시자|설명|
    |:--:|:--:|
    |%d|10진(decimal) 정수의 형식으로 출력|
    |%x|16진(hexa-decimal) 정수의 형식으로 출력|
    |%f|부동 소수점(floating-point)의 형식으로 출력|
    |%c|문자(character)로 출력|
    |%s|문자열(string)로 출력|

#### 13. printf를 이용한 출력 예제 `2-9`
- 실수형 값의 출력에 사용되는 지시자
  - `%f`: 부동 소수점의 형식으로 출력 (소수점 아래 6자리까지)
  - `%e`: 지수 형태로 출력
  - `%g`: 값을 간략하게 표현
- 전체 자리수와 소수점 아래의 자리수 지정하기
  - `%전체자리.소수점아래자리f`
  - 소수점도 한 자리를 차지하며, 소수점 아래의 빈자리는 0으로 채우고 정수의 빈자리는 공배그로 채워서 전체 자리수를 맞춘다.
  ```java
  // 예시
  
  System.out.printf("d=14.10f%n", d); // 전체 14자리 중 소수점 아래 10자리
  ```
- 지시자 '`%s`'에 숫자를 추가하여 문자열의 출력 길이 조절하기
  - 지정된 숫자부터 문자열의 길이가 작을 경우, 빈자리는 공백으로 출력
  - 공백이 있는 경우, 기본적으로 우측 끝에 문자열을 붙인다. BUT, `-`를 붙이면 좌측 끝에 붙인다.
  - `.` 붙일 경우, 문자열의 일부만 출력 가능
  ```java
  // 예시
  
  System.out.printf("[%s]%n", url); // 문자열의 길이만큼 출력공간 확보
  System.out.printf("[%20s]%n", url); // 최소 20글자 출력공간 확보 (우측정렬)
  System.out.printf("[%-20s]%n", url); // 최소 20글자 출력공간 확보 (좌측정렬)
  System.out.printf("[%.8s]%n", url); // 왼쪽에서 8글자만 출력
  ```

#### 15. 정수형의 오버플로우
- 오버플로우?
  - 타입이 표현할 수 있는 값의 범위를 넘어서는 것
  - 정수형 타입이 표현할 수 있는 최대값에 1을 더하면 최소값이 되고, 최소값에서 1을 빼면 최댓값이 된다. (ex. 계수기, 자동차 주행표시기 등)
  ```java
  최대값 + 1 → 최소값
  최소값 - 1 → 최대값
  ```

#### 16. 부호있는 정수의 오버플로우 `2-11`
- 부호 없는 정수와 비교하면 부호 있는 정수는 표현번위 즉, 최대값과 최소값이 다르기 때문에 오버플로우 발생 시점 또한 다름
- 부호 없는 정수
  - 2진수로 '0000'이 될 때 오버플로우 발생
  - `4 bit` 표현범위가 '0 ~ 15'이므로 이 값이 무한 반복
- 부호 있는 정수
  - 부호비트가 0에서 1이 될 때 오버플로우 발생
  - `4 bit` 표현범위가 '-8 ~ 7'이므로 이 값이 무한 반복

#### 17. 타입 간의 변환방법 `2-12`
  1. 숫자를 문자로 변환 - 숫자에 '0'을 더한다.
      - (char)(3 + '0') → '3'
  2. 문자를 숫자로 변환 - 문자에서 '0'을 뺀다.
      - '3' - '0' → 3
  3. 숫자를 문자열로 변환 - 숫자에 빈 문자열("")을 더한다.
      - 3 + "" → "3"
  4. 문자열을 숫자로 변환 - `Integer.parseInt()` 또는 `Double.parseDouble()`을 사용한다.
      - Integer.parseInt("3") → 3
      - Double.parsedouble("3.14") → 3.14
  5. 문자열을 문자로 변환 - `charAt(0)`을 사용한다.
      - "3".charAt(0) → '3'
  6. 문자를 문자열로 변환 - 빈 문자열("")을 더한다.
      - '3' + "" → "3"

#### 02. 연산자의 종류
- 피연산자의 개수에 따라 연산자는 '단항/이항/삼항 연산자'로 불린다.
- |종류|연산자|설명|
  |:--:|:--:|:--:|
  |산술 연산자|+ = * / % << >>|사칙 연산과 나머지 연산(%)|
  |비교 연산자|> < >= <= == !=|크고 작음과 같고 다름을 비교|
  |논리 연산자|&& || ! & | ^ ~|'그리고(AND)'와 '또는(OR)'으로 조건을 연결|
  |대입 연산자| = |우변의 값을 좌변에 저장|
  |기타|(type) ?: instanceof|형변환 연산자, 삼항 연산자, instanceof 연산자|
  
#### 03. 연산자의 우선순위
- 식에 사용된 연산자가 둘 이상인 경우, 연산자의 우선순위에 의해서 연산순서가 결정된다.
- 우선 순위: 산술 연산자 > 비교 연산자 > 논리 연산자 > 대입 연산자

#### 04. 연산자의 결합규칙  `3-1`
- 산술 > 비교 > 논리 > 대입. 대입은 제일 마지막에 수행된다.
- 단항(1) > 이항(2) > 삼항(3). 단항 연산자의 우선순위가 이항 연산자보다 높다.
- 단항 연산자와 대입 연산자를 제외한 모든 연산의 진행방향은 왼쪽에서 오른쪽이다.

#### 05. 증감 연산자 ++과 --  `3-2`, `3-3`
- 증가 연산자(++): 피연산자의 값을 1 증가시킨다.
- 감소 연산자(--): 피연산자의 값을 1 감소시킨다.
- |타입|설명|사용 예|
  |:--:|:--:|:--:|
  |전위형|값이 참조되기 `전에` 증가|j = ++i;|
  |후위형|값이 참조된 `후에` 증가|j = i++;|
```java
// 전위형
j = ++i;
++i;  // 증가 후에
j = i;  // 참조하여 대입

// 후위형
j = i++;
j = i;  // 참조하여 대입 후에
i++;  // 증가
```

#### 06. 부호 연산자 `3-4`
- 부호 연산자(-): 피연산자의 부호를 반대로 변경한 결과를 반환
  - 피연산자가 음수면 양수, 양수면 음수가 연산의 결과가 된다.
- boolean형과 char형을 제외한 기본형에만 사용 가능

#### 07. 형변환 연산자  `3-5`
- 형변환(casting): 변수 또는 상수의 타입을 다른 타입으로 변환하는 것
- 그저 피연산자의 값을 읽어서 지정된 타입으로 형변환하고 그 결과를 반환할 뿐이므로, 피연산자의 값은 형변환 후에도 아무런 변화가 없다.
- 방법: 형변환하고자 하는 변수나 리터럴의 아페 변환하고자 하는 타입을 괄호와 함께 사용 `(타입)피연산자`
- 다양한 예시:
  |변 환|수 식|결 과|
  |:--:|:--:|:--:|
  |int → char|(char)65|'A'|
  |char → int|(int)'A'|65|
  |float → int|(int)1.6f|1|
  |int → float|(float)10|10.0f|

#### 08. 자동 형변환
- 형변환 생략 시, 컴파일러가 생략된 형변환을 자동적으로 추가
  - 기존의 값을 최대한 보존할 수 있는 타입으로 자동 형변환된다.
  - 그래서 표현범위가 좁은 타입에서 넓은 타입으로 형변환하는 경우에는 값 손실이 없으므로, 두 타입 중에서 표현범위가 더 넓은 쪽으로 형변환한다. (반대의 경우 반드시 형변환 연산자를 써야 한다.)

#### 09. 사칙 연산자 `3-6`
- 올바른 연산결과를 얻기 위해서는, 피연산자들의 타입을 올바르게 지정해주거나 혹은 형변환을 해주어야 한다.

#### 10. 산술 변환  `3-7`, `3-8`, `3-9`, `3-10`
- 연산 직전에 발생하는 자동 형변환
- 산술 변환 규칙
  1. 두 피연산자의 타입을 같게 일치시킨다. (보다 큰 타입으로 일치)
    - 피연산자의 값손실을 최소화하기 위함
  ```java
  long    + int   → long    + long    → long
  float   + int   → float   + float   → float
  double  + float → double  + double  → double
  ```
  2. 피연산자의 타입이 int보다 작은 타입이면 int로 변환된다.
    - int보다 작은 타입(예: char나 short)의 표현범위가 좁아서, 연산 중에 오버플로우(overflow)가 발생할 가능성이 높기 때문
  ```java
  byte + short → int + int → int
  char + short → int + int → int
  ```

#### 11. Math.round()로 반올림하기  `3-11`
- `Math.round()` 소수점 첫째 자리에서 반올림한 결과를 정수로 반환
```java
long result = Math.round(4.52); // reulst에 5가 저장된다.
```
- 만일 소수점 첫째 자리가 아닌 다른 자리에서 반올림을 하려면 10의 n제곱으로 적절히 곱하고 나누어야 함

#### 12. 나머지 연산자  `3-12`, `3-13`
- 왼쪽의 피연산자를 오른쪽 피연산자로 나누고 난 나머지 값을 결과로 반환
- 나머지 연산자(%)는 나누는 수로 음수도 허용하나, 부호는 무시되므로 결과는 음수의 절대값으로 나눈 나머지와 결과가 같다.
- 주로 짝수, 홀수 또는 배수 검사 등에 用

#### 13. 비교 연산자
- 두 피연산자를 비교하는 연산자이며, 주로 조건문과 반복문의 조건식에 用
- 연산결과는 오직 `true`와 `false` 둘 중의 하나
- |대소비교 연산자|등가비교 연산자|
  |:--:|:--:|
  |< > <= >=|== !=|

#### 14. 문자열의 비교  `3-14`
- 문자열 비교 시, 비교연산자 `==` 대신 `equals()` 메서드 用
- `equalsIgnoreCase()`: 대소문자 구별하지 않고 비교

#### 15. 논리 연산자 && || ! `3-15`
- `&& (AND 결합)` 피연산자 양쪽 모두 true여야 true를 결과로 얻는다. `||`보다 우선순위가 높으므로, 괄호가 없다면 `||`보다 `&&`를 먼저 연산한다.
- `|| (OR 결합)` 피연산자 중 어느 한 쪽이 true면 true를 결과로 얻는다.
- |x|y|x \|\| y|x && y|
  |:--:|:--:|:--:|:--:|
  |true|true|true|true|
  |true|false|true|false|
  |false|true|true|false|
  |false|false|false|false|
- 예시
```java
① x는 10보다 크고, 20보다 작다. (10 < x && x < 20)
② i는 2의 배수 또는 3의 배수이다. (i%2==0 || i%3==0)
③ i는 2의 배수 또는 3의 배수지만 6의 배수는 아니다. ((i%2==0 || i%3==0) && i%6!=0)
④ 문자 ch는 숫자('0'~'9')이다. ('0' <= ch && ch <= '9')
⑤ 문자 ch는 대문자 또는 소문자이다. (('a' <= ch && ch <= 'z') || ('A' <= ch && ch <= 'Z'))
```

#### 16. 논리 부정 연산자  !  `3-16`
- `!` 피연산자가 true면 false를, false면 true를 결과로 반환
- 주로 조건문과 반복문의 조건식에 用
- 예시
  ```java
  // 문자 ch는 소문자가 아니다
  ch < 'a' || ch > 'z'  // 안 좋은 예
  !('a' <= ch && ch <= 'z') // 좋은 예
  
  // !!b 평가 과정
  !!b → !!true → !false → true
  ```

#### 17. 조건 연산자 `3-17`
- 조건식, 식1, 식2 모두 세 개의 피연산자를 필요로 하는 삼항 연산자
- `(조건식) ? 식1 : 식2` 조건식의 평가결과가 true면 식1이, false면 식2가 연산결과가 된다.
```java
// 조건 연산자
result = (x > y) ? x : y ;  // 괄호 생략 가능

// if문
if (x > y)
  result = x; // x > y가 true일 때
else
  result = y; // x > y가 false일 때
```

#### 18. 대입 연산자
- 변수와 같은 저장공간에 값 또는 수식의 연산결과를 저장하는데 사용
- 연산자들 중에서 가장 낮은 우선순위이므로, 식에서 제일 나중에 수행됨
- 연산 진행 방향은 오른쪽에서 왼쪽 (`x=y=3;`인 경우, `y=3` 먼저 수행 후 `x=y` 수행
- 피연산자 종류
  - `lvalue` left value, 왼쪽 피연산자. 반드시 변수처럼 값을 변경할 수 있는 것이어야 함
  - `rvalue` right value, 오른쪽 피연산자. 변수뿐만 아니라 식이나 상수 등이 모두 가능

#### 19. 복합 대입 연산자
- 대입 연산자가 다른 연산자(op)와 결합하여 `op=`와 같은 방식으로 사용되는 것
- |op=|=|
  |:--:|:--:|
  |i += 3;|i = i + 3;|
  |i -= 3;|i = i - 3;|
  |i \*= 3;|i = i \* 3;|
  |i /= 3;|i = i / 3;|
  |i %= 3;|i = i % 3;|
  |i <<= 3;|i = i << 3;|
  |i >>= 3;|i = i >> 3;|
  |i &= 3;|i = i & 3;|
  |i ^= 3;|i = i ^ 3;|
  |i \|= 3;|i = i \| 3;|
  |i \*= 10 + j;|i = i \* (10 + j);|
